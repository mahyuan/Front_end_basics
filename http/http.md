# http

- 浏览器输入Url之后发生了什么
1.redirect跳转
2.app cache 应用缓存
2.dns查找，域名解析
3.创建TCP链接
4.发送请求
5.接收响应

6.解析响应体
7.生成DOM树
8.计算DOM树上的CSS属性
9.根据CSS属性对元素逐个进行渲染，得到内存中的位图
10.一个可选的步骤是对位图进行合成，会极大地增加后续回执的速度
11.合成之后，再绘制到界面上

- 网络模型
应用层 => 传输层 => 网络层 => 数据链路层 =>  物理层 => （客户端）
物理层 => 数据链路层 => 网络层 => 传输层 => 应用层 =>  （服务端）
